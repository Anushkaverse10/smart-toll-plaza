# 🚗 AI Toll Surveillance System

A smart toll plaza monitoring system that tracks vehicle activity, alerts, and analytics through an interactive dashboard.

## 🔥 Features
- Live dashboard monitoring
- Camera surveillance interface
- Notification system
- Admin panel
- Map tracking

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript
